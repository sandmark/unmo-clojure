# Unmo - 形態素解析ベースの日本語チャットボット

Unmo は日本語チャットボットプログラムです。

## インストール

実行には JVM（Java 仮想マシン）が必要です。
https://java.com/ja/download/ からダウンロード・インストールしてください。

Unmo 本体は https://github.com/sandmark/unmo-clojure/ から
ダウンロードすることができます。
